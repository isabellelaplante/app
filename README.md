## App Description and Why I Chose This 
This app will help users budget their money efficiently by allowing them to set personal financial goals across multiple categories. my hope is that the user will be able to set a specific spending amount (like a limit) for each month. they will be able to track their expenses for the month, adding it all up at the end and will be able to see which categories they spent less in and saved, and which categories they spent more than expected so they can improve for the future.  As a junior, I got a kitchen this year which allowed me to cook my own meals instead of relying on the Tully. while I like this much better, I have come to realize that buying my own groceries is quite expensive so I would like to keep track of my spending. As a college student myself, I think a budget will be helpful in helping me keep track of what I spend and help other users as well. it will be good to start budgeting now because it will be important for when I'm out of college and on my own. 
 
## Intended Users
This app is intended for college students looking to start a budget (could eventually be expanded to people out of college looking to budget as well)
 
## Planned Features
- tracking spending through multiple categories
- creating budgeting goals for each month (and for the overall year)
- spending goal + amount actually spent (for each category and overall spending)
- graphs for a better visualization of amount spent by allowing them to set personal goals
